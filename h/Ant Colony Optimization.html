Ant Colony Optimization (ACO)

Let me break down this Ant Colony Optimization (ACO) code in very simple language, section by section.

🐜 What is Ant Colony Optimization?

Imagine real ants searching for food. They leave pheromone trails on paths they walk. Other ants follow stronger pheromone trails (shorter, better paths). Over time, the best path gets reinforced. ACO mimics this behavior to solve optimization problems.

📦 Imports

import numpy as np
import random
  • numpy → used for arrays and matrix math
  • random → used to make random choices (like an ant randomly picking a starting city)

🗺️ Distance Matrix

dist = np.array([
    [0, 2, 2, 5, 7],
    [2, 0, 4, 8, 2],
    [2, 4, 0, 1, 3],
    [5, 8, 1, 0, 2],
    [7, 2, 3, 2, 0]
])

This is a 5×5 table showing distances between 5 cities (A, B, C, D, E).

  • dist[0][1] = 2 → distance from City A to City B is 2
  • dist[0][0] = 0 → distance from City A to itself is 0 (diagonal is always 0)
  • The matrix is symmetric → A→B = B→A = 2
     A  B  C  D  E
A  [ 0, 2, 2, 5, 7 ]
B  [ 2, 0, 4, 8, 2 ]
C  [ 2, 4, 0, 1, 3 ]
D  [ 5, 8, 1, 0, 2 ]
E  [ 7, 2, 3, 2, 0 ]

🧪 Parameters

n = len(dist)          # n = 5 (number of cities)
pheromone = np.ones((n, n))  # 5x5 matrix filled with 1s
  • n → total number of cities = 5
  • pheromone → tracks how much pheromone is on each path. Starts at 1 everywhere (equal chance for all paths initially)
alpha = 1      # how much ants care about pheromone
beta = 1       # how much ants care about distance
evaporation = 0.5   # how fast pheromone fades (50% fades each round)
Q = 100        # reward amount — how much pheromone a good route gets
  • High alpha → ants follow pheromone more (trust other ants)
  • High beta → ants prefer shorter distances (be logical)
  • Evaporation = 0.5 → after each round, 50% of all pheromone disappears (prevents old bad routes from dominating forever)

📏 route_length() Function

def route_length(route):
    return sum(
        dist[route[i]][route[i+1]]
        for i in range(len(route)-1)
    ) + dist[route[-1]][route[0]]

Calculates the total distance of a route (including return to start).

Example: If route = [0, 2, 3, 4, 1] (A→C→D→E→B→back to A)

dist[0][2] + dist[2][3] + dist[3][4] + dist[4][1] + dist[1][0]
=    2     +     1      +     2      +     2      +     2      = 9
  • route[i] → current city
  • route[i+1] → next city
  • dist[route[-1]][route[0]] → return trip: last city back to first city

🧭 choose_next() Function

This is the brain of each ant — how it decides where to go next.

def choose_next(city, visited):
  • city → where the ant currently is
  • visited → list of cities already visited (can't go there again)
    probs = []
    for j in range(n):
        if j not in visited:

Loop through all cities. Skip already-visited ones.

            tau = pheromone[city][j]**alpha
  • tau = pheromone level on the path from current city to city j
  • Raised to power alpha (controls how much pheromone matters)
  • High pheromone = more attractive path
            if dist[city][j] != 0:
                eta = (1 / dist[city][j])**beta
            else:
                eta = 0
  • eta = heuristic value = 1/distance
  • Shorter distance → bigger eta → more attractive
  • If distance is 0 (same city), eta = 0 (ignore it)
  • Raised to power beta (controls how much distance matters)
            probs.append((j, tau*eta))

Attractiveness of city j = pheromone × heuristic (tau × eta)

    total = sum(p for _, p in probs)
    r = random.random()   # random number between 0 and 1
    s = 0
    for j, p in probs:
        s += p / total    # convert to probability (0 to 1)
        if r <= s:
            return j      # choose this city

This is Roulette Wheel Selection:

Cities:      B        D        E
Attraction:  3        1        6     → Total = 10
Probability: 30%      10%      60%

[----B(30%)----][--D(10%)--][-------E(60%)-------]
0              0.3         0.4                    1.0
         ↑
     r = 0.25  → picks B

A random number r lands somewhere on this wheel. Better cities have bigger slices → more likely to be picked, but not guaranteed.

🐜 aco() — Main ACO Function

def aco(iterations=50, ants=10):
  • Run 50 rounds (iterations)
  • Each round, 10 ants build routes
    best_route = None
    best_length = float('inf')  # infinity — any real route will be better

Track the best route found so far.

    for _ in range(iterations):       # repeat 50 times
        all_routes = []

        for _ in range(ants):         # send out 10 ants
            route = [random.randint(0, n-1)]   # ant starts at a random city

Each ant starts at a random city (0 to 4).

            while len(route) < n:    # keep going until all cities visited
                next_city = choose_next(route[-1], route)
                route.append(next_city)

Ant keeps choosing the next unvisited city using choose_next(), until all 5 cities are in the route.

            all_routes.append(route)
            length = route_length(route)

            if length < best_length:
                best_length = length
                best_route = route

Save the route. If it's the shortest so far, update our best.

        # Evaporation
        pheromone[:] *= (1 - evaporation)   # = pheromone * 0.5

All pheromone fades by 50%. This prevents stagnation — old bad paths lose their influence over time.

        # Pheromone update
        for route in all_routes:
            lenght = route_length(route)    # ⚠️ typo: 'lenght' instead of 'length'
            for i in range(n-1):
                pheromone[route[i]][route[i+1]] += Q / length   # ⚠️ uses outer 'length', not 'lenght'

Reward good routes with more pheromone:

  • Q / length → shorter route = more pheromone deposited
  • Example: Route of length 10 → deposits 100/10 = 10 units
  • Example: Route of length 50 → deposits 100/50 = 2 units
⚠️ Bug in original code: There's a typo lenght (unused), and Q/length uses the variable from the outer loop (last ant's length) instead of each route's own length. Should be Q/lenght or fixed to Q/route_length(route).

🏁 Output

route, length = aco()
print("Best route:", route)        # e.g. [0, 2, 3, 4, 1]
print("Shortest distance:", length) # e.g. 9

Prints the best path found and its total distance.
